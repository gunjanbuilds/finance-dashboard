#Finance Dashboard

A modern and responsive **Finance Dashboard** built using React and Tailwind CSS.
This application allows users to manage transactions, track expenses, and gain insights into their financial activity.

---

##  Live Demo

🔗 https://finance-dashboard-dun-chi.vercel.app/

---

##  Features

* ➕ Add new transactions (Admin only)
* ❌ Delete transactions
* 🔍 Search transactions by category
* 🎯 Filter by Income / Expense
* 👤 Role-based access (Admin / Viewer)
* 💾 Local Storage persistence (data saved on refresh)
* 📊 Insights section:

  * Highest spending category
  * Total transactions
  * Net balance calculation

---

##  Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS
* **State Management:** React Hooks (useState, useEffect)
* **Storage:** Browser Local Storage
* **Deployment:** Vercel

---

##  Installation & Setup

Clone the repository:

```bash
git clone https://github.com/gunjanbuilds/finance-dashboard.git
cd finance-dashboard
```

Install dependencies:

```bash
npm install
```

Run the app locally:

```bash
npm run dev
```
